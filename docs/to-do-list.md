# Infrastructure Implementation & Learning Roadmap

## [Step 1] 실무 시나리오 기반 응용 및 자동화

- [ ] **크론탭(Cron) 실행 결과 알림 시스템 구축**: 작업 성공/실패 여부 로그 기록 및 실패 시 알림 쉘 스크립트 작성
- [ ] **자동 백업 및 보관 주기 관리 스크립트**: `tar` 압축 및 `cron` 주기적 백업, `find` 활용 일정 기간 경과 파일 자동 삭제
- [ ] **디스크 모니터링 스크립트**: `df -h` 분석 기반 임계치(90%) 초과 시 경고 메시지 출력 및 관리자 알림
- [ ] **SSH 보안 강화 적용**: 포트 변경, Root 로그인 차단, SSH 키 기반 인증 설정 및 관리
- [ ] **사용자 권한 정기 점검**: 불필요 계정 정리 및 주요 설정 파일 권한(`chmod`, `chown`) 체크 스크립트 작성
- [ ] **전체 태스크 현황 리포트 자동화**: 매일 새벽 `crontab`, `at`, `ps` 정보 취합 및 Slack/Email 발송 시스템 구축

## [Step 2] 핵심 개념 재정립 및 트러블슈팅

- [ ] **RAID 장애 복구 실습**: 디스크 고장 시나리오 기반 신규 디스크 추가 및 리빌딩(Rebuilding) 절차 수행
- [ ] **LVM 볼륨 축소**: 데이터 손실 없는 안전한 볼륨 축소(Reduce) 과정 습득
- [ ] **커널 컴파일 상세 옵션**: `make menuconfig` 핵심 옵션(드라이버, 네트워크 프로토콜 등) 구분 및 설정

## [Step 3] 시스템 운영 최적화 및 심화 탐구

- [ ] **`netplan`과 `ufw/iptables` 상세 연동**: IP 설정, 특정 대역 차단, 포트 포워딩 등 네트워크 정책 설정
- [ ] **프로세스 관리와 시그널**: `SIGTERM(15)` 활용 안정적 종료 및 좀비 프로세스 대응
- [ ] **심볼릭 링크를 활용한 버전 관리**: 링크 교체 방식의 무중단 업데이트 구현 패턴 적용
- [ ] **Logrotate 활용**: 로그 자동 압축/순환/삭제 설정을 통한 디스크 점유 최적화

## [Step 4] 기술 스택 확장 및 미래 로드맵

- [ ] **엔서블(Ansible)을 이용한 서버 자동화**: 다수 가상머신 관리를 위한 프로비저닝 도구 학습
- [ ] **도커(Docker) 컨테이너 관리 전략**:
  - [ ] 컨테이너 업데이트 방식(apt update vs 이미지 빌드) 차이점 및 실무 적용 연구
  - [ ] **데이터 영속성 실습**: 바인드 마운트(Bind Mount) 및 네임드 볼륨을 이용한 데이터 보호
  - [ ] **네트워크 고도화**: 브리지 네트워크 및 MacVLAN을 통한 호스트 네트워크 연동
  - [ ] **형상 관리 및 배포**: `docker commit` 기반 골든 이미지 생성 및 Docker Hub/사설 저장소 배포
  - [ ] **환경 최적화**: 호스트-컨테이너 간 타임존(Timezone) 동기화 및 로그 타임스탬프 일치화
  - [ ] 도커 베이스 이미지 보안 취약점 점검 도구(Trivy 등) 활용
- [ ] **기업형 사설 레지스트리(Harbor) 구축**:
  - [ ] Harbor 설치 및 UI 기반 프로젝트/이미지 관리 실습
  - [ ] 이미지 취약점 자동 스캔 및 Role 기반 접근 제어(RBAC) 설정
- [ ] **보안 자동화 (ClamAV 등)**: 바이러스 스캔 결과 리포트 자동 생성 및 의심 파일 격리 프로세스 구축
- [ ] **웹 서버 성능 최적화**: Nginx 리버스 프록시, 캐싱 설정 튜닝 및 실시간 로그 분석 도구 적용

## [Step 5] 보안 및 가용성 고도화 (Starter Kit 핵심)

- [ ] **부하 테스트 및 리소스 제어 (stress)**: CPU/Memory 부하 시뮬레이션 및 cgroups 기반 자원 할당량 제한 실습
- [ ] **애플리케이션 성능 프로파일링**: APM 도구(Pinpoint, Scouter 등) 활용 병목 지점 분석 및 튜닝 전략 수립
- [ ] **auditd 보안 감사 설정**: 중요 파일 변조 감시 규칙 설정 및 로그 추적 실습
- [ ] **Nginx 로드 밸런싱 기초**: 다중 서버 환경 대비 리버스 프록시 기반 부하 분산 설정
- [ ] **서비스 상태 감시 및 셀프 힐링**: 프로세스 중단 감지 시 자동 재기동 쉘 스크립트 구현
- [ ] **pre-commit 및 Gitleaks 설정**: 로컬 커밋 단계에서의 시크릿 유출 자동 차단 환경 구축
- [ ] **더미 웹서버 보안 점검 (SCA)**: `npm/pnpm audit`을 활용한 오픈소스 라이브러리 취약점 검사 자동화
- [ ] **Trivy 이미지 보안 스캔**: 도커 이미지 빌드 후 OS 및 패키지 취약점 정밀 스캔 절차 구현

## [Step 6] 엔터프라이즈 고가용성 및 분산 스토리지 (HA & Storage)

- [ ] **Ceph 클러스터 구축**: Proxmox 연동 분산 스토리지 구성 및 데이터 삼중화 검증
- [ ] **Galera Cluster & ProxySQL**: DB 멀티 마스터 복제 및 쿼리 기반 부하 분산 처리
- [ ] **Redis Sentinel 및 분리 설계**: 세션/캐시용 Redis 분리 및 장애 조치(Failover) 자동화
- [ ] **pfSense 보안 게이트웨이**: 인프라 전면 방화벽 구축 및 서브넷 격리 정책 적용

## [Step 7] 클라우드 네이티브 관측성 및 AI 자동화 (Advanced)

- [ ] **Thanos & Loki 통합 모니터링**: 멀티 클러스터 데이터 통합 및 중앙 집중식 로그 분석
- [ ] **k6 부하 테스트 및 p95 분석**: 시스템 한계 측정 및 지연 시간 기반 성능 리포트 생성
- [ ] **Argo CD를 이용한 GitOps 구현**: 선언적 상태 동기화 및 지속적 배포 자동화
- [ ] **KEDA 및 AI 부하 예측 확장**: 이벤트 기반 오토스케일링 및 AI 모델 연동 선제적 자원 증설
