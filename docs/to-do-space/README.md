# 📂 생성된 학습 공간 구조 (to-do-space)

- **step1_automation:** 크론탭 알림, 백업, 디스크 모니터링, SSH 보안, 사용자 권한 감사 수행
- **step2_troubleshooting:** RAID 장애 복구, LVM 볼륨 축소, 커널 컴파일 옵션 학습
- **step3_optimization:** 네트워크/방화벽 연동, 프로세스 시그널 관리, 심볼릭 링크 버전 관리, Logrotate 최적화
- **step4_expansion:** Ansible 입문, Docker 관리 전략, ClamAV 보안 자동화, Nginx 성능 튜닝
- **step5_advanced_security:** auditd 감사, 로드 밸런싱, 셀프 힐링, Gitleaks, 취약점 스캔 (Starter Kit 핵심)
- **step6_enterprise_ha:** Ceph 분산 스토리지, Galera DB Cluster, Redis Sentinel, pfSense 보안 게이트웨이
- **step7_cloud_native_ai:** Thanos/Loki 통합 관측성, k6 부하 테스트, Argo CD GitOps, AI 기반 예측 오토스케일링

1. 각 폴더 README.md 파일의 이론 파악
2. 제공된 스크립트 파일을 가상머신(VM)에서 직접 실행 및 검증
