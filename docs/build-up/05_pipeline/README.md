# [Phase 5] Pipeline: 보안 자동화 및 이미지 관리

공급망 보안 강화 및 신뢰성 있는 배포 파이프라인 구축 절차

---

## 1. 기업형 이미지 관리 (Harbor)

- **Harbor Registry 구축:** 사설 레지스트리 설치 및 프로젝트별 권한(RBAC) 설정
- **이미지 보안 스캔:** Harbor 내장 스캔 엔진을 통한 취약점 분석 및 위험 이미지 배포 차단
- **가비지 컬렉션:** 미사용 이미지 자동 정리를 통한 스토리지 용량 최적화

## 2. 로컬 및 CI 보안 가드레일

- **pre-commit 설정:** 커밋 단계에서의 스타일 교정(Prettier) 및 구문 검사 강제화
- **Gitleaks 도입:** 소스 코드 내 시크릿(API Key, Password) 유출 탐지 및 원천 차단
- **ShellCheck 연동:** 쉘 스크립트의 논리 오류 및 보안 결함 자동 진단

## 3. 지속적 통합 보안 (CI Security)

- **SAST (Semgrep):** 정적 분석 도구를 통한 코드 내 보안 취약 패턴 실시간 스캔
- **SCA (pnpm audit):** 오픈소스 라이브러리의 알려진 취약점(CVE) 점검 및 업데이트 유도
- **Trivy 검사:** 최종 이미지 아티팩트 및 OS 패키지의 보안 결함 최종 검증
