# Cloud Infra Solutions

본 디렉토리는 클라우드 인프라 구축 및 운영에 필수적인 4가지 핵심 오픈소스 솔루션을 다룹니다.

### 솔루션 구성

1. **01_Airflow**: 인프라 작업의 자동화 및 워크플로우 관리 (Data Orchestration)
2. **02_Wazuh**: 실시간 위협 감지, 규정 준수 및 취약점 분석 (Security XDR/SIEM)
3. **03_Prometheus_Grafana**: 지표 수집 및 시각화를 통한 시스템 모니터링 (Monitoring)
4. **04_Vault**: 인프라 전반의 기밀 정보 및 접근 권한 중앙 관리 (Secret Management)

### 상업적 이용 및 라이선스 확인 (오픈소스 기반)

- **Apache Airflow**: Apache License 2.0 (상업적 이용 가능)
- **Wazuh**: Apache License 2.0 (상업적 이용 가능)
- **Prometheus & Grafana**: Apache License 2.0 / AGPL v3 (오픈소스 에디션은 상업적 이용 가능)
- **HashiCorp Vault**: BSL (Community Edition은 상업적 이용 시 특정 조건 확인 필요, 일반적 오픈소스 용도로 사용 가능)
