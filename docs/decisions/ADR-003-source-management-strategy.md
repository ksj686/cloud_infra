# ADR-003: 소스 관리: 분석용 소스 경로 최적화

- **Status:** ✅ Accepted
- **Date:** 2026-04-01
- **Decider:** Gemini CLI & User

## Context (배경)

- 강의 자료 및 분석용 텍스트 파일(`lecture/`)이 프로젝트 루트에 산재하여 디렉토리 구조가 복잡해짐.
- 루트 디렉토리는 핵심 설정 파일과 문서 중심으로 정결하게 유지될 필요가 있음.

## Decision (결정)

- 모든 분석용 소스 및 강의 자료를 **`docs/lecture/`** 하위로 이동하여 관리함.
- 단, MkDocs 웹 포털 메뉴에서는 제외(`exclude_docs`)하여 실제 문서와 분석 자료를 논리적으로 분리함.

## Consequences (결과)

- **이득:** 프로젝트 루트 구조의 정결성 확보 및 문서 기반의 통합 관리 가능.
- **비용:** MkDocs 설정(`mkdocs.yml`) 내 경로 관리 및 제외 규칙 적용 오버헤드 발생.
