# 02. 다계층 보안 방어 체계 (Security Guardrails)

### 🛡️ Shift-Left 보안 전략

- **Gitleaks:** API Key, Secret Token 유출 원천 차단
- **ShellCheck:** 쉘 스크립트 문법 사전 진단 및 논리 버그 탐지
- **환경 무결성:** LF 강제 및 BOM 제거를 통한 Win-Linux 실행 호환성 보장
- **Prettier:** 일관된 스타일 가이드를 통한 가독성 표준화

### 🔒 공급망 보안 및 시스템 감사

- **SCA (Dependency Audit):** 오픈소스 라이브러리의 알려진 취약점(CVE) 전수 검사
- **Harbor Registry:** 사설 저장소 기반의 이미지 권한 관리 및 보안 스캔 통합
- **auditd:** 커널 레벨의 파일 변조 실시간 감시 및 모든 명령어 실행 이력 추적
