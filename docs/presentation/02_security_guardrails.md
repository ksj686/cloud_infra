# 02. 다계층 보안 방어 체계 (Security Guardrails)

## 🛡️ Shift-Left 보안 전략
- **개념:** 보안 검증을 운영 단계가 아닌 개발 단계(커밋 전)로 전진 배치
- **핵심 도구:**
    - **Gitleaks:** 시크릿 유출 자동 차단
    - **ShellCheck:** 쉘 스크립트 문법 및 로직 오류 사전 진단
    - **Prettier:** 문서 및 설정 파일의 일관된 스타일 강제

## 🔒 공급망 보안 및 감사
- **SCA (Dependency Audit):** 오픈소스 라이브러리의 알려진 취약점 정밀 탐지
- **auditd:** 커널 레벨의 파일 변조 및 명령어 실행 이력 추적성 확보
