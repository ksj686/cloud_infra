# 03. 무중단 가용성 아키텍처 (High Availability)

### ⚖️ 서비스 연속성 확보

- **DB 고가용성(HA, Galera Cluster):** 3노드 멀티 마스터 구성을 통한 무중단 데이터베이스 서비스
- **하이퍼바이저 고가용성(HA, Proxmox):** 물리 노드 장애 시 가상 머신(VM) 자동 이동 및 복구
- **MacVLAN:** 컨테이너에 호스트 네트워크 대역 IP를 직접 할당하여 통신 효율성 극대화

### 📂 데이터 가용성 고도화

- **분산 스토리지 (Ceph):** 네트워크 기반 삼중화 저장을 통한 하드웨어 결함 방어
- **시간 동기화 (Timezone Synchronization):** 호스트-컨테이너 간 시간 동기화를 통한 로그 무결성 보장
