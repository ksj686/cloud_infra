# 03. 무중단 가용성 아키텍처 (High Availability)

## ⚖️ 서비스 연속성 확보
- **DB HA (Galera Cluster):** 3노드 멀티 마스터 구성을 통한 무중단 DB 서비스 및 데이터 실시간 동기화
- **하이퍼바이저 HA (Proxmox):** 물리 서버 장애 시 가상 머신의 자동 이동(Migration) 및 복구

## 📂 데이터 가용성 고도화
- **분산 스토리지 (Ceph):** 네트워크 기반 삼중화 저장을 통한 하드웨어 결함 완전 방어
- **ProxySQL:** 지능형 부하 분산을 통한 DB 성능 최적화 및 안정성 유지
