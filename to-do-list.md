## [Step 1] 실무 시나리오 기반 응용 및 자동화

- [x] **자동 백업 및 보관 주기 관리 스크립트**: `tar`로 주요 데이터를 압축하고 `cron`으로 주기적 백업 수행, `find` 명령어를 이용해 일정 기간(예: 7일)이 지난 백업 파일 자동 삭제

- [x] **SSH 보안 강화 적용**: 기본 포트 변경, Root 로그인 차단, 패스워드 인증 대신 SSH 키 기반 인증 설정 및 관리
- [x] **사용자 권한 정기 점검**: `/etc/passwd`, `/etc/group` 파일을 확인하여 불필요한 계정을 정리하고, 주요 설정 파일의 권한(`chmod`, `chown`)이 올바른지 체크하는 스크립트

- [x] **커널 컴파일 상세 옵션**: `make menuconfig` 시 수많은 옵션 중 실무에서 커스텀이 필요한 핵심 옵션(드라이버, 네트워크 프로토콜 등) 구분법
